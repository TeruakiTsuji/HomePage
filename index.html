Wine 6.0 Source Download and Compile

1. yumリポジトリにEPELを追加

# yum install epel-release

# yum install wine --enablerepo=epel

2. 64bit版wineのuninstall

# yum remove wine

# yum erase wine wine-*

3. wineのビルド

# yum install samba-winbind-clients -y

# yum -y groupinstall base "Development tools" --setopt=group_package_types=mandatory,default,optional

# yum -y install libjpeg-turbo-devel libtiff-devel freetype-devel

# yum -y install glibc-devel.{i686,x86_64} libgcc.{i686,x86_64} libX11-devel.{i686,x86_64} freetype-devel.{i686,x86_64} gnutls-devel.{i686,x86_64} libxml2-devel.{i686,x86_64} libjpeg-turbo-devel.{i686,x86_64} libpng-devel.{i686,x86_64} libXrender-devel.{i686,x86_64} alsa-lib-devel.{i686,x86_64}

○wineのソースをダウンロード

# cd /usr/src

# wget http://dl.winehq.org/wine/source/6.0/wine-6.0.tar.xz

# tar Jxfv wine-6.0.tar.xz

○ビルド(64bit版と32bit版)

# cd wine-6.0/

# mkdir -p wine32 wine64

# cd wine64

# ../configure --enable-win64

# make -j 2

# cd ../wine32

# ../configure --with-wine64=../wine64

# make -j 2

# make install

# cd ../wine64

# make install

4. cabextractのinstall

# yum --enablerepo=epel install cabextract

(実際にはcabextractのソースのダウンロードとビルドを行う)

5. winetricksのinstall

# mkdir bin

# cd bin

# wget https://raw.githubusercontent.com/Winetricks/winetricks/master/src/winetricks

# chmod +x winetricks

6. winetricksでallfontsのinstall

# winetricks allfonts
